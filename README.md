# 🧮 Simple Calculator (Python)

A beginner-friendly Python calculator that takes a full expression in one line and evaluates it like a real calculator.

---

## 🚀 Features
- One-line input (e.g. `20+5*2`)
- Supports:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Basic input validation
- Simple and clean code for beginners
- ## 🧑‍💻 Example
- Enter expression: 20/5
4.0
